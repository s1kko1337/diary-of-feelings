# Diary of Feelings -- Полная Концепция Продукта

> Версия 2.0 -- Полный рестарт, 2026-03-07

---

## Суть продукта

Личное пространство для ежедневной рефлексии, эмоциональной поддержки и осознанного планирования. Два крыла одного приложения: **Задачи** (действия, планы, знания) и **Эмоции** (чувства, терапия, поддержка). Между ними -- AI-помощник, который видит обе стороны жизни пользователя.

Никакого давления. Никаких метрик ради метрик. Пространство, куда хочется возвращаться.

---

## UX-флоу

### 1. Splash Screen

- Полноэкранный сплэш при запуске
- Плавная волновая SVG-анимация (liquid / wave effect)
- Логотип и название приложения
- Автоматический переход на главную (1.5--2 секунды)

### 2. Главная страница

- Крупный приветственный текст: **"Привет! Как ты сегодня?"**
- Интерактивный **древовидный граф** (SVG/Canvas) с двумя ветками:
  - **Задачи** -- левая ветка (индиго/синие тона)
  - **Эмоции** -- правая ветка (лаванда/розовые тона)
- Клик по ветке -> плавный свайп-переход на соответствующий раздел
- Граф анимирован: мягкое покачивание веток, пульсация узлов
- Минимальный интерфейс -- только дерево и приветствие

### 3. Раздел ЗАДАЧИ

Свайп вправо от главной или клик по ветке "Задачи".

**Bottom navigation** раздела: Задачи | Заметки | Библиотека

#### 3.1 Задачи (TasksView)

Визуальный планировщик на основе календаря и стикеров.

- **Гридовый календарь** текущего месяца
- **3 типа стикеров** (цветные карточки):
  - Красный (#ffd0cc) -- срочное / важное
  - Жёлтый (#fff3b0) -- обычное
  - Зелёный (#d4f0d4) -- приятное / награда
- **Drag & drop**: стикер можно зажать и перетащить на дату
- При перетаскивании / клике открывается **"бумажное" окно стикера**:
  - Текст задачи
  - Категория (работа / личное / здоровье / развитие)
  - Уведомления вкл/выкл
- Стикеры на датах отображаются как маленькие цветные метки
- Paper texture эффект на стикерах (лёгкая текстура бумаги)

#### 3.2 Заметки (NotesView)

Минималистичный блокнот в стиле iOS Notes.

- Список заметок с превью первой строки
- Inline-создание новой заметки
- Содержимое парсится AI-агентом для персонализации советов
- Поддержка базового форматирования (заголовки, списки)
- Поиск по заметкам

#### 3.3 Библиотека (LibraryView)

Личная база знаний пользователя.

- Загрузка и хранение документов (Word, PDF, статьи)
- Превью содержимого
- Теги и категории для организации
- Поиск по библиотеке
- AI может ссылаться на материалы из библиотеки в советах

### 4. Раздел ЭМОЦИИ

Свайп влево от главной или клик по ветке "Эмоции".

**Bottom navigation** раздела: Эмоции | Помощник | Отчёт

#### 4.1 Эмоции (EmotionsView)

Объединяет дневник эмоций и КПТ-дневник.

**Дневник эмоций:**
- Колесо эмоций -- двухуровневый выбор точной эмоции
- Интенсивность (шкала 0--10)
- Одна строка в день -- минимальный барьер входа
- Тайм-капсула -- запись, которую увидишь через 30 дней
- История записей с визуализацией

**КПТ-дневник (СМЭР) -- Kanban board:**
- Grid board с drag & drop карточками
- Колонки по этапам проработки или по датам
- Каждая карточка -- запись СМЭР:
  - С -- Ситуация (объективно)
  - М -- Мысли (автоматические)
  - Э -- Эмоции (с интенсивностью)
  - Р -- Реакция (поведение)
- Продвинутый режим: когнитивные искажения, оспаривание, альтернативная мысль
- Перетаскивание карточек между состояниями (новая -> в работе -> проработана)

#### 4.2 Помощник (AssistantView)

AI-чат с персональным агентом.

- Полноценный чат-интерфейс (пузыри сообщений)
- Агент опрашивает пользователя о настроении
- Даёт советы на основе:
  - Текущих эмоций и истории
  - Записей КПТ-дневника
  - Задач и заметок
  - Материалов из библиотеки
- Предлагает статьи и упражнения
- Тон: тёплый, между хорошим другом и дзен-монахом
- Формат: короткий абзац + одно конкретное действие

#### 4.3 Отчёт (ReportView)

Аналитика эмоционального состояния.

- Генерируется на основе всех данных пользователя
- Выбор периода (неделя / месяц / произвольный)
- Графики динамики эмоций
- Обнаруженные паттерны (AI)
- Прогресс по КПТ-записям
- Корреляции: задачи <-> эмоции

## Навигация

### Структура

```
[Splash] -> [Home: Tree Graph]
                |
        +-------+-------+
        |               |
    [ЗАДАЧИ]        [ЭМОЦИИ]
    Bottom Nav:     Bottom Nav:
    - Задачи        - Эмоции
    - Заметки       - Помощник
    - Библиотека    - Отчёт
```

### Переходы

- Главная <-> Разделы: **свайп-анимация** (горизонтальная)
- Внутри раздела: bottom nav переключает вкладки (fade transition)
- Назад на главную: кнопка "домой" в header или свайп
- Нет сайдбаров, нет верхнего pill-навбара (заменён на tree graph)
- Контент по центру, `max-width: 720px`

### Профиль и настройки

- Доступ через аватар/иконку в header
- Тема (светлая/тёмная), уведомления, сброс данных

## Онбординг

Лёгкий 5--7 шаговый флоу при первом входе:

1. Имя пользователя
2. Что сейчас даётся тяжело?
3. Чего хочешь от этого пространства?
4. Утренний или вечерний человек?
5. Какие темы близки? (отношения, работа, смысл, здоровье, рост)
6. Как комфортнее получать обратную связь?

Профиль обновляется автоматически по мере использования.

## Принципы UX

- **Никакого давления** -- нет агрессивных стриков, нет failure states
- **Минимальный барьер входа** -- одна строка уже запись
- **Тёплый тон** -- не клинический, не корпоративный
- **Два крыла** -- действия и чувства связаны, но не смешаны
- **AI как друг** -- не врач, не коуч, а тёплый собеседник
- **Paper & organic** -- стикеры, бумажные текстуры, природные формы дерева

## Дизайн

### Философия

Сочетание modern minimal с organic / paper элементами. Чистый интерфейс с тактильными акцентами (стикеры, бумажные текстуры, древовидный граф).

### Палитра

| Назначение | Цвет | Hex |
|------------|-------|-----|
| Primary (индиго) | Основные действия, ветка задач | Из дизайн-спеки |
| Лаванда | Ветка эмоций, фоны | #c4b5e0 |
| Персик | Акценты, тепло | #f8c9a0 |
| Мята | Успех, позитив | #a8dfc8 |
| Роза | Эмоции, внимание | #f0b8c0 |
| Стикер красный | Срочные задачи | #ffd0cc |
| Стикер жёлтый | Обычные задачи | #fff3b0 |
| Стикер зелёный | Приятные задачи | #d4f0d4 |

### Типографика

- Inter -- основной шрифт
- Крупный приветственный текст на главной (2.5--3rem)
- Мягкие, округлые формы

### Анимации

- Splash: волновая SVG-анимация (liquid wave)
- Tree graph: мягкое покачивание, пульсация узлов
- Переходы: горизонтальный свайп между разделами
- Стикеры: drag preview с тенью и наклоном
- Фоновые орбы: медленный drift (20--25s)
- Hover: мягкий подъём карточек
- Все анимации спокойные, без резких движений
