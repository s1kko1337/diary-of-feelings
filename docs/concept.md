# Diary of Feelings — Концепция продукта

> Версия 2.0, 2026-03

---

## Суть продукта

Личное пространство для ежедневной рефлексии, эмоциональной поддержки и осознанного планирования. Два крыла одного приложения: **Задачи** (действия, планы, знания) и **Эмоции** (чувства, терапия, поддержка). Между ними — AI-помощник, который видит обе стороны жизни пользователя и формирует персональный эмоциональный портрет.

Никакого давления. Никаких метрик ради метрик. Пространство, куда хочется возвращаться.

## UX-флоу

### 1. Splash Screen

Полноэкранный сплэш при запуске — волновая SVG-анимация, логотип. Автопереход через 1.5–2 секунды.

### 2. Главная страница

- Крупный приветственный текст с обращением по имени и фразой дня
- Интерактивный **древовидный граф** (SVG) с двумя ветками: Задачи (индиго) / Эмоции (лаванда)
- Ветки показывают число активных элементов и текущую эмоцию
- Граф анимирован: мягкое покачивание, пульсация узлов
- **Эмоциональный портрет** — стресс, тренд настроения, продуктивность
- **Рекомендации дня** — 3 статьи (утро / день / вечер), подобранные под портрет

### 3. Навигация

Единая нижняя панель с 5 вкладками:

```
[ Задачи ]  [ Эмоции ]  [ 🏠 ]  [ Помощник ]  [ Отчёт ]
```

Кнопка "Домой" (центр) — elevated circle. Нет сайдбара, нет двух отдельных nav.

### 4. Раздел ЗАДАЧИ (`/tasks?tab=`)

Единый экран с тремя свайпаемыми вкладками:

#### Задачи

Визуальный планировщик на основе календаря и стикеров.

- **Гридовый календарь** текущего месяца, свайп по месяцам
- **3 типа стикеров**: красный (#ffd0cc) — срочное, жёлтый (#fff3b0) — обычное, зелёный (#d4f0d4) — приятное
- **Drag & drop** стикеров на даты (desktop + touch)
- Стикер: текст, completed-чекбокс, приоритет, время
- **DateTasksPanel** — список задач при тапе на дату

#### Заметки

Минималистичный блокнот в стиле iOS Notes.

- Список карточек с 2-строчным превью, относительной датой, pin и цветом (6 вариантов)
- Inline-редактор с автосохранением (debounce 1.5s)
- AI анализирует тексты заметок для построения эмоционального портрета

#### Библиотека

Личная база знаний: документы, PDF, статьи с тегами и поиском.

### 5. Раздел ЭМОЦИИ (`/emotions`)

Два раздела переключаются вкладками внутри страницы:

#### Дневник эмоций

- **Колесо эмоций** — двухуровневый выбор точной эмоции
- **Интенсивность** 0–10
- **Одна строка в день** — минимальный барьер
- **Тайм-капсула** — запись, которую увидишь через 30 дней
- **Timeline** история с фильтрами (неделя / месяц / по эмоции)
- **Стрик** и недельный мини-граф

#### КПТ-дневник (СМЭР) — Kanban board

Три колонки: Новая → В работе → Проработана. Drag & drop карточек.

**Структура карточки СМЭР:**

| Поле | Описание |
|------|----------|
| С — Ситуация | Что произошло объективно |
| М — Мысли | Автоматические мысли |
| Э — Эмоции | Эмоция + интенсивность 0–10 |
| Р — Реакция | Поведение в ситуации |

**Продвинутый режим** (toggle): когнитивное искажение, оспаривание, альтернативная мысль, эмоции после проработки.

Карточка показывает прогресс-бар заполнения, тег искажения, подсветку если не трогали 3+ дня.

### 6. Помощник (`/assistant`)

AI-чат с персональным агентом.

- Полноценный chat-интерфейс с typewriter-стримингом
- 5 стартовых чипов-подсказок
- Агент знает: имя, текущие эмоции и историю 7 дней, задачи, КПТ-карточки, эмоциональный портрет
- Тон: тёплый, между хорошим другом и дзен-монахом
- Формат ответа: короткий абзац + одно конкретное действие

### 7. Отчёт (`/report`)

- Выбор периода (неделя / месяц / 3 месяца)
- Тепловая карта настроения
- Bar chart топ эмоций
- AI-инсайты: паттерны, лучший день недели, корреляции задач и эмоций

## Эмоциональный портрет и рекомендации

Все действия пользователя — эмоции, задачи, КПТ-записи, заметки — формируют **эмоциональный портрет**:

| Параметр | Как формируется |
|----------|-----------------|
| Доминирующая эмоция | Топ за 30 дней с exponential decay |
| Тренд настроения | Сравнение rolling 3-day avg (последние 3 дня vs дни 4–7) |
| Уровень стресса | Медиана интенсивности негативных эмоций за 7 дней |
| Продуктивность | Выполненные / всего задач за 30 дней |
| Топ искажения | Частота КПТ-искажений `{distortion: count}` |
| Активные темы | Ключевые слова из ситуаций КПТ + текстов заметок |

**3 рекомендации в день** (утро / день / вечер) подбираются на основе портрета:
- При стрессе ≥ 7/10 — только статьи для высокого стресса (calming/grounding)
- Тональность статьи матчится к тренду настроения (declining → calming, improving → motivating)
- Фидбек (👍/👎) накапливается и учитывается в завтрашнем подборе

## Онбординг

5–7 шагов при первом входе:

1. Имя
2. Что сейчас даётся тяжело?
3. Чего хочешь от этого пространства?
4. Утренний или вечерний человек?
5. Какие темы близки?
6. Как комфортнее получать обратную связь?

## Принципы UX

- **Никакого давления** — нет агрессивных стриков, нет failure states
- **Минимальный барьер** — одна строка уже запись
- **Тёплый тон** — не клинический, не корпоративный
- **AI как друг** — не врач, не коуч
- **Органика** — стикеры, бумажные текстуры, дерево

## Дизайн

### Философия

Modern minimal + organic paper elements. Glassmorphism на навигации, paper-текстуры на стикерах, SVG-граф с анимацией.

### Палитра

| Назначение | Hex |
|------------|-----|
| Primary (indigo) | `#6366f1` |
| Лаванда | `#c4b5e0` |
| Персик | `#f8c9a0` |
| Мята | `#a8dfc8` |
| Роза | `#f0b8c0` |
| Стикер красный | `#ffd0cc` |
| Стикер жёлтый | `#fff3b0` |
| Стикер зелёный | `#d4f0d4` |

### Анимации

- Splash: волновая SVG (liquid wave), GSAP
- Tree graph: мягкое покачивание, пульсация узлов (GSAP, CSS)
- Переходы в TasksView: CSS slide-left / slide-right
- Стикеры: drag preview с тенью и наклоном
- Hover: 0.2–0.3s ease. Page transitions: 0.4–0.6s ease-out. Нельзя анимировать `backdrop-filter`.
